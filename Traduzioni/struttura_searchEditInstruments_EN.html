SEARCH AND EDIT INSTRUMENTATION

To edit an instrumentation, you have to find it on our Database; to do this you can use various search criteria:

  • "By name" search: every instrumentation with that name will be shown;
  • "By Exact Daily Cost" search: every instrumentation with the exact cost given will be shown. Cost parameter needs to be typed numerically (example: 15);
  • "By Minimum Daily Cost" search: every instrumentation with a cost equal or greater than the given one will be shown. Cost parameter needs to be typed numerically (example: 15);
  • "By Maximum Daily Cost" search: every instrumentation with a cost equal or lower than the given one will be shown. Cost parameter needs to be typed numerically (example: 15);
  • "By Maximum Availability" search: every instrumentation with maximum availability given will be shown. Availability parameter needs to be typed numerically (example: 15);

Click now on "Search" button to display results.
