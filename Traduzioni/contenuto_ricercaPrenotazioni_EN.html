FIND AND MODIFY A BOOKING

To modify a booking previously made by a user it's necessary to search it in the database. You can use different search criterions:

  • If you search "By Name" every booking made by the username inserted in "Search Term" will be shown;
  • If you search "By Date" every booking in the date inserted in "Search Term" field will be shown. The date must be inserted in "dd/mm/yyyy" format;
  • If you search "By Room" every booking of the inserted room in "Search Term" field will be shown;
  • If you search "By Function" every booking for the function inserted in "Search Term" field will be shown.

After that, click on "Search" to start searching.
