SEARCH AND EDIT ROOMS

To edit a room, you have to find it on our Database; to do this you can use various search criteria:

  • "By name" search: every room with that name will be shown;
  • "By Function" search: every room with that purpose will be shown
  • "By Exact Hourly Cost" search: every room with the exact cost given will be shown. Cost parameter needs to be typed numerically (example: 15)
  • "By Minumim Hourly Price" search: every room whit a cost equal or greater than the given one will be shown. Cost parameter needs to be typed numerically (example: 15)
  • "By Maximum Hourly Cost" search: every room whit a cost equal or lower than the given one will be shown. Cost parameter needs to be typed numerically (example: 15)

Click now on "Search" button to display results.
