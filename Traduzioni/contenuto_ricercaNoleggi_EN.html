FIND A RENTAL

In order to modify a rental made by a user it's necessary to search it in the database before. It's possible to use various search criteria:

  • If you search "By name" every rental made by the username inserted will be shown;
  • If you search "By rented instrumentation" every rental concerning the inserted instrumentation will be shown;
  • If you search "By rental started after" every rental that starts after the inserted date will be shown. The date must be inserted in dd/mm/yyyy format;
  • If you search "By rental ended by" every rental that will end by inserted date will be shown. The date must be inserted in dd/mm/yyyy format;
  • If you search "By rental duration" every rental that will last the inserted days will be shown. The number of days must be inserted in numeric format (e.g. 3).

Click on "Search" button to start searching.
